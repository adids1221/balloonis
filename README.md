## Backend
### API
#### Packages
1. GET /api/packages
2. GET /api/packages/:id
3. POST /api/packages
4. DELETE /api/packages/:id

#### Balloon Designs
1. GET /api/balloons
2. GET /api/balloons/:id
3. POST /api/balloons
4. DELETE /api/balloons/:id

#### Wall Tattoos
1. GET /api/tattoos
2. GET /api/tattoos/:id
3. POST /api/tattoos
4. DELETE /api/tattoos/:id
